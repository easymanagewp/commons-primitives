1. INTRODUCTION

The primitives Component contains a set of Java classes for working with primitive types.

2. DEPENDENCIES

The primitives component is dependent upon the following external components for development and use:

  • Java Development Kit (Version 1.2 or later)
  • Collections Testing Framework (Version 3.0) - for unit tests only, not required for deployment
  • JUnit Testing Framework (Version 3.7 or later) - for unit tests only, not required for deployment

3. RELEASE INFO

Current Release: primitives is yet to be released.

Planned Next Release - 1.0. This release is intended to formally release the code previously located in commons-collections component. For backwards compatability, the original collections package names will be used.
